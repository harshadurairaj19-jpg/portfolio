Selected Work

PROJECT — 01  ·  FEATURED

CADAI — AI-Powered EEG Cognitive Workload Classification for Aviation Safety

End-to-end deep learning system classifying pilot cognitive workload (GO / CAUTION / NO GO) from 32-channel EEG signals using a CNN+GCN hybrid model achieving 88.89% LOSO-CV accuracy across 10 subjects. Integrated WGAN-GP augmentation (63→900 samples), SHAP explainability, TF-IDF+FAISS RAG pipeline with DeepSeek-R1 7B for real-time advisory generation, and a React HUD dashboard for live instructor-pilot monitoring.

CNN + GCNWGAN-GPEEG 32-ch SHAPFAISS RAGDeepSeek-R1 7B Flask APIReact + Vite
PROJECT — 02

Trajectory Tracking & Obstacle Avoidance using TurtleBot3 Burger

Feedback-controlled navigation system enabling TurtleBot3 Burger to track straight-line, circular, and pentagonal trajectories while avoiding static and dynamic obstacles. Implemented a hybrid Artificial Potential Field (APF) + Bug2 algorithm for smooth reactive avoidance with reliable path recovery, validated first in ROS2/Gazebo/RViz simulation and then deployed on real hardware, with tracking accuracy evaluated using IAE, ISE, and MSE error metrics.

ROS2GazeboRViz Feedback ControlAPFBug2 Algorithm TurtleBot3 BurgerOdometry
PROJECT — 03

Adaptive Exoskeleton with EEG–EMG Fatigue Detection

Real-time ML pipeline fusing EEG, EMG, and text embeddings. A deep CNN + logistic regression model generates a live fatigue score (0–100) to dynamically adjust exoskeleton support levels.

Deep CNNEEG / EMGSignal ProcessingML
PROJECT — 04

Autonomous Room-Level Delivery Robot

Indoor delivery robot using Monte Carlo Localization (MCL) for precise room-level navigation and real-time obstacle avoidance in dynamic environments, built on a ROS2 framework.

ROS2Monte Carlo LocalizationAutonomous Nav

FARM RUN — AI Grid Adventure Game
